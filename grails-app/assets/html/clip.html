Using this extension, you can clip, annotate, tag, and share any web page within the Bloomberg Terminal without actually bringing up the tTrminal.
Logging into the Bloomberg terminal requires going through four factor authentication. This requires quite a bit of fime and effort which you might not want to go through if you want to take a simple note from a web page. If your'e an analyst writing a research article, this comes up often.
This was impacting some of the projects I was working on, so I decided to write the extension. A few month's later I had Bloomberg's first in house built chrome extension on the web store.

- javascript, jQuery, sass, groovy, grails, python, C++
It's on chrome and firefox
